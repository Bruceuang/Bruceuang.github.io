Which x for that sum?

level

5kyu

题目描述

这是一道数学题，有函数 U(n, x) = x + 2x**2 + 3x**3 + .. + nx**n , 其中(0 < x < 1) ， 当 n 趋向于正无穷时，函数的极限为 m ；求解，当已知极限 m 时，x 的值为多少。

Examples:

solve(2.0) returns 0.5 ， 当 n 趋于无穷大时，极限 m == 2.0 ，返回 x 的值 0.5.

解题思路

函数U(n, x) = x + 2x**2 + 3x**3 + .. + nx**n ，有无穷多项无法计算，我们要将其简化成一个能求解的式子

这样，当已知 m 时，就能求出 x
